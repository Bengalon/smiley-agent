# אתרים לעסקים מקומיים · גוש דן

פרויקט שמאתר עסקים מקומיים בגבעתיים / תל אביב / רמת גן **שאין להם אתר אינטרנט**,
מרכז את הפרטים בקובץ אקסל, ובונה לכל עסק **אתר מלא, נגיש ומותאם אישית** (HTML/CSS/JS, RTL).

## מה יש כאן

| קובץ / תיקייה | תיאור |
|---|---|
| `data/businesses.json` | 20 העסקים שנאספו — שם, קטגוריה, עיר, כתובת, טלפון, מקור, סטטוס אתר |
| `build.py` | מחולל האתרים + עמוד האינדקס + קובץ האקסל |
| `verify.py` | בדיקת רינדור, גלישה אופקית ונגישות בסיסית (דורש playwright) |
| `leads.xlsx` | גיליון הלידים המסודר (2 גיליונות: לידים + סיכום) |
| `sites/index.html` | עמוד אינדקס שמקשר לכל 20 האתרים |
| `sites/<slug>/index.html` | אתר עצמאי לכל עסק — ניתן לפריסה כמו שהוא |

## 20 העסקים (8 תחומים · 3 ערים)

מספרות ועיצוב שיער · מכולות ומינימרקטים · פלאפל וחומוס · חנות פרחים ·
סנדלרות ותיקונים · קוסמטיקה ומניקור · מאפיות בורקס · חייטות ותיקוני בגדים.

כל העסקים נבחרו מפני שהם מופיעים **רק במדריכים / Waze / פייסבוק** — כלומר אין
להם אתר ייעודי משלהם. הפירוט המלא ב-`leads.xlsx` ובעמודת "סטטוס אתר".

## מה כל אתר כולל

- **RTL מלא בעברית**, עיצוב מותאם לקטגוריה עם ערכת צבעים ייחודית לכל עסק
- כותרת (header) דביקה עם ניווט ותפריט מובייל (המבורגר)
- Hero עם כפתורי פעולה: התקשרות (`tel:`), וואטסאפ (`wa.me`), ניווט (Google Maps / Waze)
- מקטעי שירותים/תפריט, יתרונות, אודות, גלריה, המלצות, שעות פעילות ופרטי קשר
- **טופס יצירת קשר** עם ולידציה בצד לקוח והודעת אישור נגישה (`aria-live`)
- **נגישות:** דילוג-לתוכן, landmarks סמנטיים, `:focus-visible`, ניגודיות,
  `prefers-reduced-motion`, h1 יחיד, שדות טופס עם `label`
- **SEO:** תגיות meta/OG + נתוני `schema.org/LocalBusiness` (JSON-LD)
- קובץ בודד, ללא תלות בקבצים חיצוניים — נטען מהר ועובד גם אופליין

## בנייה מחדש

```bash
pip install -r requirements.txt
python3 build.py
```

בדיקה אוטומטית (אופציונלי):

```bash
pip install playwright
python3 verify.py /tmp/shots
```

## פריסה

כל תיקיית עסק היא אתר סטטי עצמאי. אפשר להעלות את `sites/` כמו שהוא ל-Netlify /
Vercel / GitHub Pages, או כל שרת סטטי. `sites/index.html` הוא עמוד הנחיתה.

## הערה חשובה על הנתונים

הפרטים (שם, כתובת, טלפון) נאספו **ממקורות פומביים** (מדריכי עסקים, מפות, רשתות חברתיות)
ומשקפים את מה שנמצא בחיפוש. **מומלץ לאמת כל פרט מול העסק לפני פנייה או פרסום**, במיוחד
מספרי טלפון וכתובות. השדות שלא נמצאו סומנו "לאימות". שעות הפעילות, המחירים,
התמונות וההמלצות בכל אתר הם **תוכן לדוגמה** שהעסק אמור להחליף בתוכן אמיתי משלו.
